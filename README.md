# GraphConnect

> Analyze social networks through graph theory.

GraphConnect is a web application for exploring and analyzing social network data. Upload your network, visualize relationships, detect communities, and identify influential users using graph theory metrics—all in your browser.

**Live Demo:** https://graphconnect-nu.vercel.app

## Features

- **Interactive Visualization**  
  Explore graphs with pan, zoom, and search.

- **Community Detection**  
  Automatically identify densely connected groups.

- **Centrality Analysis**  
  Find influential users using Degree, Betweenness, and Closeness centrality.

- **Graph Theory Metrics**  
  Analyze graph density, shortest paths, bridge nodes, and connected components.

- **Fast Uploads**  
  Upload CSV edge lists and analyze networks in seconds.

- **Own Your Data**  
  Every project is securely scoped to your account.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/graphconnect.git
cd graphconnect
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Start the development server:

```bash
npm run dev
```
